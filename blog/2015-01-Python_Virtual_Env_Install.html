安装过程中可能遇到的问题

有时候，一处小小的错误就会让你折腾好几天。我遇到的一些问题如下：

(1)设置Python系统环境变量：在系统变量中新建一项，变量名为Python，变量值为D:/Python/Python27。

Note

这里一定要注意，变量值后面千万不要加分号，否则在powershell或者cmd中运行python命令会找不到。

(2)Python在安装的时候，千万不要安装到一个带空格的目录里面，比如不要安装在C:/Program Files/Python27，否则后面执行C:/Program Files/Python27/Scripts中的pip，

virtualenv会出现错误。

Note

这个装到类似D:/Python/Python27的目录下就好了，这个地方折腾了我好几天。

安装过程中的几个主要的命令

打开PowerShell，输入：

Start-Process powershell -Verb runas

在新打开的PowerShell中输入：

Set-ExecutionPolicy RemoteSigned

下一步回车即可。

下面的命令用来完成virtualenv和virtualenvwrapper-powershell的安装设置：

pip install virtualenv

pip install virtualenvwrapper-powershell

mkdir $env:WORKON_HOME

cd $env:WORKON_HOME

import-module virtualenvwrapper

创建新的Python虚拟环境

mkvirtualenv rosenv --no-site-packages
