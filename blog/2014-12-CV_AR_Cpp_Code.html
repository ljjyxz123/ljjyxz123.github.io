各种跟CV、AR相关的C/C++代码收集

  • Tue, 16 Dec 2014
  • Brad Lucas
  • Computer Vision
  • CV, AR, Cpp, Code,

这个页面力图搜集各种跟CV，AR相关的代码，如无特别声明，均是c/c++代码。还是一贯的标准，不求全面，只求质量。如有特别推荐的代码，请在本页留言，或者email我：cvchina AT gmail.com

欢迎转载或参考，但请尊重作者劳动成果，转载请注明出处，参考请引用链接！

通用库/General Library

OpenCV: http://opencv.willowgarage.com/ 无需多言。

RAVL Recognition And Vision Library. 线程安全。强大的IO机制。包含AAM。

http://www.ee.surrey.ac.uk/CVSSP/Ravl/

CImg 很酷的一个图像处理包。整个库只有一个头文件。包含一个基于PDE的光流算法。

http://cimg.sourceforge.net/

图像，视频IO/Image, Video IO

FreeImage: http://freeimage.sourceforge.net/ DevIL: http://openil.sourceforge.net/ ImageMagick: http://www.imagemagick.org/script/index.php FFMPEG: http://www.ffmpeg.org/ VideoInput: http://www.muonics.net/school/spring05/videoInput/ portVideo: http://portvideo.sourceforge.net/

AR相关/Augmented Reality

ARToolKit: http://www.hitl.washington.edu/artoolkit/ 基于Marker的AR库

ARToolKitPlus: http://handheldar.icg.tugraz.at/artoolkit ARToolKit的增强版。实现了更好的姿态估计算法。

PTAM: http://www.robots.ox.ac.uk/~gk/PTAM/ 实时的跟踪、SLAM、AR库。无需Marker，模板，内置传感器等。

BazAR: http://cvlab.epfl.ch/software/bazar/inde 基于特征点检测和识别的AR库。

局部不变特征/Local Invariant Feature

VLFeat: http://www.vlfeat.org/ 目前最好的Sift开源实现。同时包含了KD-tree，KD-Forest，BoW实现。

Ferns: http://cvlab.epfl.ch/software/ferns/index.php 基于Naive Bayesian Bundle的特征点识别。高速，但占用内存高。

SIFT By Rob Hess: http://blogs.oregonstate.edu/hess/cod 基于OpenCV的Sift实现。

目标检测/Object Detection

AdaBoost By JianXin.Wu: http://c2inet.sce.ntu.edu.sg/Jianxin/RareEvent/rare_event.htm 又一个AdaBoost实现。训练速度快。

行人检测 By JianXin.Wu: http://c2inet.sce.ntu.edu.sg/Jianxin/projects/Pedestrian/Pedestrian.html 基于Centrist和Linear SVM的快速行人检测。

（近似）最近邻/ANN

FLANN: http://www.cs.ubc.ca/~mariusm/index.p 目前最完整的（近似）最近邻开源库。不但实现了一系列查找算法，还包含了一种自动选取最快算法的机制。

ANN: http://www.cs.umd.edu/~mount/ANN/ 另外一个近似最近邻库。

SLAM & SFM

SceneLib [LGPL]: http://www.doc.ic.ac.uk/~ajd/Scene/ monoSLAM库。由Androw Davison开发。

图像分割/Segmentation

SLIC Super Pixel: http://ivrg.epfl.ch/supplementary_material/RK_SLICSuperpixels/index.html 使用Simple Linear Iterative Clustering产生指定数目，近似均匀分布的Super Pixel。

目标跟踪/Tracking

TLD: http://info.ee.surrey.ac.uk/Personal/Z.Kalal/tld.html 基于Online Random Forest的目标跟踪算法。

KLT: http://www.ces.clemson.edu/~stb/klt/ Kanade-Lucas-Tracker

Online boosting trackers: http://www.vision.ee.ethz.ch/boostingTrackers/ Online Boosting Trackers

直线检测/Line Detection

DSCC: http://www.umiacs.umd.edu/~zhengyf/LineDetect.htm 基于联通域连接的直线检测算法。

LSD [GPL]: http://www.ipol.im/pub/algo/gjmr_line_segment_detector/ 基于梯度的，局部直线段检测算子。

指纹/Finger Print

pHash [GPL]: http://phash.org/ 基于感知的多媒体文件Hash算法。（提取，对比图像、视频、音频的指纹）

图像检索/Image Retrieval

libpmk: http://www.google.com/url? Pyramid Matching Algorithm实现。

vocsearch: http://www.inf.ethz.ch/personal/fraundof/page2.html 一个简单的的Vocabulary Tree实现。

视觉显著性/Visual Salience

Global Contrast Based Salient Region Detection: http://cg.cs.tsinghua.edu.cn/people/~cmm/saliency/ Ming-Ming Cheng的视觉显著性算法。

FFT/DWT

FFTW [GPL]: http://www.fftw.org/ 最快，最好的开源FFT。

FFTReal [WTFPL]: http://ldesoras.free.fr/prod.html#src_fft 轻量级的FFT实现。许可证是亮点。

音频处理/Audio processing

STK [Free]: https://ccrma.stanford.edu/software/stk 音频处理，音频合成。

libsndfile [LGPL]: http://www.mega-nerd.com/libsndfile/ 音频文件IO。

libsamplerate [GPL ]: http://www.mega-nerd.com/libsndfile/ 音频重采样。
